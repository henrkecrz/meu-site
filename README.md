# Nexus Commerce

Marketplace premium de tecnologia em HTML, CSS e JavaScript, com visual inspirado em e-commerces reais.

## Paginas criadas

- `index.html` — home de marketplace com busca, categorias, hero promocional, ofertas relampago, produtos, marcas e recomendacoes.
- `produtos.html` — pagina real de listagem com filtros laterais, ordenacao, chips de filtros, grid de produtos e paginacao.
- `produto.html` — pagina de detalhe com galeria, informacoes, preco, frete, quantidade e compra.
- `login.html` — pagina de login/cadastro com botoes Google, Apple e Microsoft, alem de email/senha.

## Observacao sobre login social

Os botoes de Google, Apple e Microsoft estao prontos visualmente, mas ainda sao demonstrativos. Para login real, integre Firebase Auth, Supabase Auth, Auth0 ou outro provedor OAuth.

## Como abrir

Abra `index.html` no navegador ou publique em GitHub Pages, Hostinger, Vercel ou Netlify.

## Proximos passos recomendados

1. Conectar produtos, estoque e pedidos a um backend real.
2. Ativar login social com OAuth.
3. Adicionar checkout com Pix, cartao e boleto.
4. Criar painel de vendedor e administrador em paginas separadas.
5. Integrar frete, nota fiscal e gateway de pagamento.

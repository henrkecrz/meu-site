# Nexus Commerce

E-commerce premium focado em tecnologia, criado como uma SPA estatica em HTML, CSS e JavaScript.

## Modulos incluidos

- Vitrine inicial com produtos tech em destaque
- Janela do cliente com catalogo, filtros e carrinho simulado
- Janela do vendedor com pedidos, estoque e indicadores
- Administrador geral com KPIs, governanca e alertas de risco
- Portal B2B para clientes de clientes, subcontas e aprovacoes
- Suporte/pos-venda com tickets, garantia e assistente IA simulado

## Como abrir

Basta abrir o arquivo `index.html` no navegador.

## Proximos passos recomendados

1. Integrar autenticacao real por perfil de usuario.
2. Migrar para Vue, React ou Next.js quando o projeto precisar escalar.
3. Adicionar backend com produtos, pedidos, vendedores e pagamentos reais.
4. Conectar gateway de pagamento, frete, nota fiscal e painel administrativo.
5. Publicar em Hostinger, Vercel, Netlify ou GitHub Pages.
